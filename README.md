1. Reverse a list

Exercise description:
Write a program that takes a list of integers and prints its reverse list.
Hint: Use a loop to reverse the list.

Sample input:
[1, 2, 3, 4, 5]
Sample output:
[5, 4, 3, 2, 1]

2. Remove duplicate elements from a list

Exercise description:
Write a program that takes a list of numbers and removes all duplicate elements.
Hint: Use lists and loops.

Sample input:
[1, 2, 2, 3, 4, 4, 5]
Sample output:
[1, 2, 3, 4, 5]
